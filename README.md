# Unifi.id Website

A people-first smart building and decarbonisation platform website built with Next.js, React, TypeScript, and Tailwind CSS.

## Tech Stack

- **Framework:** Next.js 16.1.4
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **React:** 19.2.3

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

### Build

```bash
npm run build
npm start
```

## Project Structure

- `/app` - Next.js app router pages and layouts
- `/components` - Shared React components (Header, Footer)
- `/src/components` - Feature components (HeroSection, Card, etc.)
- `/src/lib` - Utilities and animations
- `/src/content` - Content data

## Features

- Page transitions with Framer Motion
- Scroll-triggered animations
- Premium hover effects
- Hero motion animations
- Responsive design
- Accessibility support (reduced motion)
